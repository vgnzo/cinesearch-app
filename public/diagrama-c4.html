Nível 3 — Componentes do Backend

As rotas do server.js e o que cada uma faz

server.js — Rotas da API REST
➕
Cadastrar Filme
POST /filmes 🔒
Requer token JWT. Valida campos e título duplicado. Salva com autor_id do usuário logado.
🔍
Buscar Filmes
GET /filmes?q=termo
Pública. Usa multi_match para buscar em título, gênero e sinopse. Sem termo, retorna todos (match_all).
✏️
Editar Filme
PUT /filmes/:id 🔒
Requer token JWT. Verifica se autor_id do filme = usuário logado antes de atualizar.
🗑️
Deletar Filme
DELETE /filmes/:id 🔒
Requer token JWT. Verifica se autor_id do filme = usuário logado antes de remover.
🔐 Rotas de Autenticação
📝
Registrar Usuário
POST /usuarios/registro
Valida campos, verifica email duplicado, criptografa senha com bcrypt (salt 10) e salva no índice "usuarios".
🔑
Login
POST /usuarios/login
Busca usuário por email, compara senha com bcrypt.compare() e retorna token JWT assinado (expira em 8h).
