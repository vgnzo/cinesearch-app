Nível 3 — Componentes do Backend

As rotas do server.js e o que cada uma faz

server.js — Rotas da API REST
➕
Cadastrar Filme
POST /filmes
Recebe os dados do formulário e cria um novo documento no índice com ID gerado automaticamente pelo Elasticsearch.
🔍
Buscar Filmes
GET /filmes?q=termo
Usa multi_match para buscar em título, gênero e sinopse ao mesmo tempo. Sem termo, retorna todos (match_all).
✏️
Editar Filme
PUT /filmes/:id
Substitui o documento inteiro no Elasticsearch pelo conteúdo atualizado, identificado pelo _id único.
🗑️
Deletar Filme
DELETE /filmes/:id
Remove permanentemente o documento do índice no Elasticsearch usando o _id retornado na busca.
